# MiniSTL
  一个STL的子集，采用C++11实现STL的基本组件，各类容器和算法
* 开发语言 :  `C++11`
* 编译环境 :  `Visual Studio Community 2013`
* 参考资料1:  `STL源码剖析 侯捷著`
* 参考资料2:  `[gcc 4.9 源代码](https://github.com/gcc-mirror/gcc/tree/master/libstdc%2B%2B-v3/include)`
